下载项目后，新建数据库bookstore，然后导入bookstore.sql即可。

导入项目后,在resource文件夹下修改db.properties 将数据库账号和密码修改成自己MYSQL数据库的账号密码即可。

技术总结：<br>
  控制器：springMVC <br>
  Dependence Injection:spring<br>
  ORM:mybatis<br>
  日志：log4j<br>
  前端：Bootstrap<br>
  数据库：mysql<br>
